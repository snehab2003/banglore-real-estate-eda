# banglore-real-estate-eda
Exploratory data analysis of banglore housing prices with data cleaning,visualization and business insights using Python &amp; Tableau.This project provides an in-depth analysis of the Bangalore housing market using data visualization in Tableau. The goal is to uncover trends in availability, affordability, and built-up area distribution to help home buyers, investors, and real-estate enthusiasts understand the property landscape in Bangalore.

### Data Cleaning & Preprocessing

- Removed duplicate rows
- Cleaned inappropriate square-feet values in Excel
- Standardized location names and sorted alphabetically
- Fixed bathroom count issues using Pandas
- Converted price values to numeric type using Pandas
- Created new column: price_per_sqft = price / total_sqft
